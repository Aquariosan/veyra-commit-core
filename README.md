# Veyra

Commit mode for production AI agent actions.

Explore is free. Commit runs through Veyra.

Veyra gives AI tools and agent systems a machine-readable path into trusted production mode. Free trust discovery. Free verification. Pay only when actions settle.

## Why Veyra

Without Veyra, a productive write is just an API call.

With Veyra, every productive write can become:
- delegated
- verified
- settlement-backed
- attributable
- machine-recoverable

## Install

```bash
npm install @veyra/sdk-node
## Core idea

When a tool rejects a productive write, it can return VeyraCommitRequired.

Agents and frameworks can then:
	1.	detect the refusal
	2.	call handleCommitRequired()
	3.	obtain a commit token
	4.	retry automatically
